# Game

Текстовая игра

Название: Приключения короля Артура



Инструкция по запуску:

1\) Необходимо скачать все файлы в одну папку

2\) Запустить файл Game\_course.py



Как играть:

При запуске игры появляется главное меню. Если игрок уже зарегистрирован в игре, он может начать новую игру (цифра 1). Если нет, необходимо для начала зарегистрироваться (цифра 3). После входа в игру необходимо выбрать 3 артефакта, после чего появляется легенда и игровое меню. 

При выборе направо (цифра 1), игрок попадает к старцу (загадки). 

При выборе налево (цифра 2) - пещера с гномом (игра в кости). 

Показать статус (цифра 4) - выводятся характеристики Артура (здоровье, сила, интеллект, наличие Экскалибура и Священного Грааля, эликсира, артефакты).

Сохранить игру (цифра 5) - сохраняет игру в файл game\_save.txt.

Выйти в меню (цифра 6) - возвращает в главное меню.

Битва с Мордредом (цифра 3) - бой с боссом. Необходимо пройти отя бы одну из веток.



Персонажи:

1\) Артур - главный герой. Может атаковать, атаковать Экскалибуром, если он есть, пить Эликсир здоровья, если он есть. Проходя испытания (ветки) получает артефакты.

2\) Маг Мерлин помогает Артуру в битве с Мордредом (появляется в 3 ветке). Может атаковать огнем, лечит союзников. На действия тратится мана.

3\) Рыцарь Ласелот помогает Артуру в битве с Мордредом (появляется в 3 ветке). Может атаковать копьем. Имеет щит, который защищает его от ударов, если позволяет прочность.



Босс:

Мордред - может атаковать одного персонажа или всех сразу.



Артефакты:

1\) Меч Экскалибур (+35 к урону). Можно получить, разгадав загадки старца.

2\) Эликсир здоровья (+25 к здоровью). Можно получить, разгадав загадки старца.

3\) Священный Грааль (+20 к силе). Можно получить у гнома, выиграв его в кости.

4\) Щит (можно выбрать) уменьшает получаемый урон в 1.5 раза.

5\) Кинжалы (можно выбрать) +15 к урону

6\) Броня (можно выбрать) уменьшает получаемый урон в 2 раза

7\) Волшебная карта (можно выбрать) не дает преимущества

8\) Деревянный лук (можно выбрать) +10 к урону

9\) Компас (можно выбрать) не дает преимущества

10\) Плащ-невидимка (можно выбрать) уменьшает получаемый урон в 2.5 раза

11\) Единорог (можно выбрать) не дает преимущества



Файлы:

1\) Characters.py - файл с персонажами. Здесь находится родительский класс Human с методами получения урона и восстановления здоровья, а таже декоратор, позволяющий узнать жив герой или нет. Дочерние классы Hero (Артур), Mag (Мерлин), Knight (Ланселот) со своими методами атаки, получения урона и специальными свойствами персонажа. Функция для записи хода игры в файл result.txt.

2\) Boss.py - файл с боссом. Класс Boss, имеющий свои методы атаки и декоратор позволяющий узнать жив босс. Также функция рандомной атаки босса.

3\) Save\_and\_registration.py - файл с сохранением игры и регистрацией. Функции сохранения игры в текстовый файл game\_save.txt. А также функции регистрации нового пользователя и авторизации в игре. Имя и пароль пользователей сохраняются в файл accounts.txt.

4\) Artifacts.py - файл с артефактами. Создание копилки артефактов и сохранение их в файл artifact\_pool.txt. Выбор игроком артефактов из копилки и сохранение их в файле user\_artifacts.txt. Функция, возвращающая артефакты игрока в копилку.

5\) First\_branch.py - 1 сюжетная ветка. Встреча со старцем и загадки. Возможность получить Экскалибур и Эликсир здоровья. Ветку можно пройти неограниченное кол-во раз.

6\) Second\_branch.py - 2 сюжетная ветка. Пещера с гномом (игра в кости). Возможность получить Священный Грааль. Ветку можно пройти неограниченное кол-во раз.

7\) Battle.py - 3 сюжетная ветка (битва с боссом). Сначала ходят персонажи, потом босс. Бой заканчивается если умер босс, либо умер Артур. Ветку можно пройти только 1 раз.

8\) Game\_course.py - главный файл (запускает игру). Содержит главную функцию main.
